“종이의 집: 공동 경제구역”의 공개일에 맞추어 이 프로젝트를 진행하게 되었으며, 이 프로젝트를 통해 기획과 디자인 등 Node.js 이외의 분야도 함께 다루며 한 층 더 성장할 수 있었습니다.
</aside>
